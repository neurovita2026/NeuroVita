# 📖 About

NeuroVita is an AI-powered preventive cognitive health platform designed to help people understand how their daily lifestyle choices may influence long-term brain health.

The project combines modern web technologies with evidence-based wellness concepts to encourage healthier habits before cognitive decline develops.

Clone the repository

```bash

```

```bash

```

```text

```

or use the VS Code Live Server extension.
